# NimbusAI Data Analyst Intern Assignment

## 📌 Overview

This project analyzes customer churn, engagement, and feature usage using SQL, MongoDB, Python, and Power BI.

---

## 🛠️ Tools & Technologies

* PostgreSQL (SQL)
* MongoDB
* Python (Pandas, NumPy, SciPy)
* Power BI

---

## 📊 Key Analysis

### 1. Customer Churn Analysis

* Identified churn distribution across plan tiers
* Higher churn observed in lower engagement segments

### 2. Engagement Analysis

* Strong relationship between engagement and churn
* Highly engaged users are less likely to churn

### 3. Feature Usage

* Feature_A is the most used feature
* Indicates its importance in user retention

### 4. Activity Trend

* Declining user activity observed over time

---

## 📈 Dashboard

Power BI dashboard includes:

* Churn vs Plan Tier
* Engagement vs Churn
* Customer Segmentation
* Feature Usage
* Activity Trend

---

## 🎯 Key Recommendations

1. Improve onboarding for low-engagement users
2. Invest in high-usage features like Feature_A
3. Address declining user activity trends
